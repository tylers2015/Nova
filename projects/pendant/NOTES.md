# Nova Pendant Notes

The first version should be push-to-talk, not always listening.

Button behavior:
- Hold button: listen
- Release button: send command
- Short tap: check connection
- Double tap: stop Nova speaking
- Long hold: pairing/config mode

Status sounds:
- Single high ding: success
- Low buzz: command not understood
- Double ding: connected
- Descending tone: failed

Status LED:
- Blue pulse: idle/connected
- Yellow: waiting for Nova
- Green flash: command accepted
- Red flash: not understood
- Purple: config/pairing mode
