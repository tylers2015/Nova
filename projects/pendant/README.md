# Nova Pendant

A battery-powered wearable push-to-talk device for staying connected to Nova.

## V1 Goals
- Push-to-talk voice command button
- ESP32-S3 based
- I2S microphone
- Small status speaker for dings
- RGB status LED
- Wi-Fi connection to Nova AI PC
- Rechargeable LiPo battery
- Designed to be repairable and expandable

## Planned Hardware
- ESP32-S3
- INMP441 I2S microphone
- MAX98357A I2S amplifier
- Tiny 8 ohm speaker
- WS2812B RGB LED
- Push button
- TP4056 USB-C LiPo charger/protection board
- 1000-2000mAh LiPo battery
- Optional vibration motor
